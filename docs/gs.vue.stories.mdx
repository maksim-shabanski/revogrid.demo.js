# Getting Started
#
## Vue
Set ignore web-component as part of vuejs template: `Vue.config.ignoredElements = [/revo-\w*/];`.
Define webcomponent `<revo-grid/>` and you can access it in a "Vue way" for example as `<revo-grid :source.prop="source"/>`.
Check Show code for mode details.
